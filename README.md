# :dizzy: 프로젝트 개요

> [Linkuni]는 나중에 다시 보고싶은 링크들을 한 곳에 모아 정리하고 쉽게 찾아보기 위한 서비스입니다.

> URL을 저장할 때 달아놓은 태그를 이용해 태그별로 URL을 분리하여 키워드 검색이 가능합니다.
